# Análisis de Exoplanetas con IA

Este proyecto analiza datos reales de exoplanetas usando técnicas
de inteligencia artificial no supervisada (K-Means).

## Autor
Estudiante independiente (13 años)

## Tecnologías
- Python
- Pandas
- Scikit-learn
- Google Colab

## Objetivo
Identificar patrones y posibles anomalías en planetas fuera del sistema solar.
